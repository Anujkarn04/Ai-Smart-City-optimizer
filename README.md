# ⚡ AI Smart City Energy Optimizer

A real-time **Smart City Dashboard** that predicts energy demand, monitors zones, and optimizes resource distribution using AI-driven insights.

## 🧠 Problem Statement

Modern cities face challenges in:
- ⚡ Energy demand fluctuation  
- 🌍 Inefficient resource distribution  
- 🚨 Lack of real-time monitoring  

This project provides a **smart, data-driven solution** for energy optimization.

---

## 💡 Solution

This system:
- Predicts **energy demand in real-time**
- Visualizes **city zones and load distribution**
- Detects **critical zones & anomalies**
- Provides **optimization insights**

---

## ✨ Features

- 📊 **Energy Demand Prediction**
- 📈 **Actual vs Predicted Graph**
- 🌍 **Interactive Smart City Map**
- ⚡ **Zone Optimization Panel**
- 🌱 **Renewable Energy Insights**
- 🚨 **Anomaly Detection System**
- 🔄 **Real-time Updates**

## 🧪 How It Works

1. User inputs real-time conditions:
   - Hour, Temperature, Previous Consumption

2. Backend processes input using prediction logic

3. API returns predicted energy demand

4. Frontend visualizes:
   - Graphs
   - Zone status
   - Insights
